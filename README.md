Инструкция по установке (Windows):

Для начала установить версию python от 3.14
Проверить версию можно с помощью команды
py --version

Прописать в консоль:
1. cd ~\ComputerStoreWeb-main\src\csw_backend - перейти в папку проекта
1. py -m pip install poetry - установить poetry
3. py -m poetry install - установить зависимости
3. py -m poetry run py manage.py runserver - запустить проект
5. http://127.0.0.1:8000/ - перейсти по ссылке

Чтобы установить программы, нажмите в консоли сочетание клавиш CTRL+C
